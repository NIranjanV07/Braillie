AI-Powered Smart Braille System
Developed an intelligent assistive system that converts text, images, PDFs, and speech into tactile Braille to enhance accessibility for visually impaired and deafblind users. The system integrates OCR, speech recognition, text-to-speech, and Braille conversion with ESP32-controlled servo motors to dynamically generate Braille characters. It also features finger-gesture recognition for interactive learning, read-aloud assistance, quizzes, and real-time feedback to improve Braille literacy. A web-based interface enables users to upload documents, capture images, and interact with the system, creating an inclusive and engaging learning experience.
